# Группа 4: Игры и приложения

## 1. Игры и приложения (раздел "Контент" -> "Приложения")

Блок «Контент» позволяет удалённо добавлять ярлыки приложений на игровые станции. Доступ: Дополнительное меню -> «Контент» -> «Приложения». Кнопка «Добавить игру» в правом верхнем углу.

### Поля карточки игры/приложения
- **Название игры** (обязательное) — от 2 до 50 Unicode символов.
- **Путь к игре** (обязательное) — соответствует пути запуска на станции. Можно задать до 3-х альтернативных путей для разных залов/станций.
- **Аргумент командной строки** (опционально).
- **Рабочая директория** (заполняется автоматически после указания пути).
- **Группа аккаунтов** (доступно в тарифах STARTER и BUSINESS) — связь с клубными аккаунтами.
- **Возрастное ограничение** — запрет запуска лицам ниже рейтинга.
- **Разрешить запускать от имени администратора** (флаг).
- **Обложка игры**: мин. 256×384, оптимально 600×900, соотношение 2:3, макс. 250 КБ, форматы `.jpg`, `.png`.
- **Путь к иконке** (только для приложений) — форматы `.ico`, `.exe`.

ВАЖНО: если путь некорректный или приложение не найдено на конкретной станции — в режиме редактирования отображается значок-заглушка, клиент ярлык не видит. Шелл при каждом запуске проверяет пути и показывает только найденные приложения.

### Группы игр/приложений
Кнопка «Управление группами» в правом верхнем углу. Изменение порядка / перенос в группу — drag-and-drop левой кнопкой мыши.

## 2. Возрастной ценз
В карточке приложения включите переключатель **«Возрастное ограничение»** и выберите рейтинг (PEGI/ESRB можно посмотреть в Wikipedia, пример — Dota 2). После сохранения рейтинг отображается в карточке и в клиентском Шелле.

## 3. Добавление приложений из Microsoft Store
1. Запустить **PowerShell** на весь экран.
2. Выполнить `Get-StartApps` (или `Get-StartApps | Select-String "название*"`).
3. Скопировать AppID из правой колонки (например `33C30B79.HyperXNGenuity_0a78dr3hq0pvt!App`).
4. В поле «Путь» карточки указать: `shell:AppsFolder/<AppID>`.

## 4. Создание ярлыков из Шелла
- Войти в режим редактирования: **Ctrl+Alt+P** -> пароль по умолчанию `pasw0rd` (сменить в ПУ: Настройки -> Шелл -> Пароль высокого доступа) -> «Войти».
- Раздел «Игры» или «Приложения» -> «**+ Игра**» / «**+ Приложение**».
- Поля: Название, минимум один путь, аргумент CLI, скрипт до запуска, пароль запуска, группа клубных аккаунтов, рабочая директория (все кроме названия и пути — опционально).
- Загрузить обложку (требования те же, что в п.1) -> «Сохранить».
- Шелл синхронизирует карточки на остальные станции после перезагрузки — настраивать на каждом ПК не нужно.

## 6. Обновление игр оператором (служебный тариф)

**Создание тарифа:** Доп. меню -> «Товары и услуги» -> «Тарифы» -> «Создать». Название «Обновление игр. Служебный», длительность **1 час**, стоимость **0 руб** во всех залах, круглосуточная продажа, **отключить** отображение из Шелла.

**Действия оператора:** продать тариф на ПК как **Гостевую сессию**, запустить обновление, при необходимости продлить тем же тарифом (например, Warzone 120 ГБ), завершить гостевой сеанс.

**Контроль (менеджер):** Логи -> поиск «Служебный» или «Обновление» -> список активаций с привязкой к сотруднику. Опционально включить уведомление о применении тарифа в TG-канал.
