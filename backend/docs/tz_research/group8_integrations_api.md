# Группа 8: Интеграции, API, Аналитика, Логи

## 1. Внешние интеграции
Доступны на тарифе Business: **СБП** (оплата из шелла и мобильного приложения), **Kaspi** (Kaspi QR/онлайн для Казахстана), **CloudPayments** (онлайн-платежи клиентов), **Telegram** (отправка уведомлений). Рассылка SMS-кодов — на всех тарифах автоматически без доплаты. Для регистрации/восстановления пароля используется звонок на номер либо подтверждение через Telegram/VK.

## 2. Интеграция кассы (ККТ)
Подписка Business. Соединение через расширение **Kkm-Server** (Chrome/Edge/Firefox/Opera/Yandex) + десктоп-приложение KKMServer. Поддерживаемые протоколы ФФД 1.2: **АТОЛ** (1Ф, 11Ф, 15Ф, 20Ф, 22Ф/FPrint-22ПТК, 25Ф, 27Ф, 30Ф, 35Ф, 42ФС, 47ФА, 50Ф, 52Ф, 55Ф, 60Ф, 77Ф, 90Ф/91Ф/92Ф, Sigma 10/150Ф, Казначей ФА), **ШТРИХ-М** (М-01Ф, ON-LINE, ЛАЙТ-01Ф/02Ф, МИНИ-01Ф/02Ф, ФР-01Ф/02Ф), **RR-Electro** (РР-01Ф…04Ф), **ЭЛВЕС-ФР-Ф**, **Ритейл** (01Ф/02Ф/Комбо), **Dreamkas Viki-Print** (57Ф/57+Ф/80+Ф), **КИТ/КАСБИ**. Эквайринг: TTK2, СБРФ, INPAS (Verifone/IRAS/PAX), UCS, ARCUS-2 (Ingenico). Тип устройства — «Фискальный регистратор», COM/Ethernet. В панели: Настройки → Касса → модель, СНО, тип налога, категории продаж.

## 3. Телеграм-бот для клиентов
Бот `@smartshell_auth_bot`. Кнопки (не slash-команды): **Запустить / Авторизоваться** (отправка контакта), **Зарегистрируйте меня!** (никнейм 5–30 симв., дата рождения дд.мм.гггг, пароль 6–30), **Получить код** (код подтверждения, действует 5 минут), **Мои клубы** (список посещённых клубов с балансами).

## 4. SmartGamer (мобильное приложение клиента)
Google Play, AppStore, RuStore, NashStore. Регистрация: страна, дата рождения, никнейм, телефон, email, пароль, капча, верификация звонком (или входящий звонок/Telegram/VK). Функции: выбор и бронирование игрового места, онлайн-пополнение депозита, профиль, восстановление пароля.

## 5. SmartKiosk (киоск самообслуживания)
iOS-приложение (iOS 13+, AppStore, бесплатно). Авторизация: телефон менеджера + пароль + **ClubID** (виден в панели управления у названия подписки). Функции: регистрация клиентов (требует включения «Регистрация клиентов оператором»), пополнение депозита через СБП, карта клуба со свободными местами, витрина товаров, кастомная заставка при бездействии, правила клуба.

## 6. Контроллер SmartShell (реле питания)
Подключает PlayStation/XBOX к карте клуба. Требования: 2× HDMI ≥2.1 суммарно ≤3.5 м, Wi-Fi 2.4 ГГц. Подключение: запрос уникального **KEY** у техподдержки, разборка корпуса, удержание кнопки на плате, подключение питания в **Input**, Wi-Fi-сеть **ESPconfig** (пароль `espconfig`) → `192.168.4.1` → ввод SSID/PASSWORD/KEY (без спецсимволов, английская кодировка) → Submit → Reboot. ТВ подключается в **Output**.

## 7. SmartShell API
**GraphQL** (Query, Mutation) — клиент запрашивает только нужные поля одним запросом. Документация и примеры: `https://apidoc.smartshell.gg/index.html`. Авторизация и токен оформляются через панель управления, эндпоинт доступен внешним системам интеграции.

## 8. Аналитика (отчёты)
Доступ: `admin.smartshell.gg/reports`. Отчёты: **по сменам** (выручка по сменам/операторам, часы, бонусы, возвраты), **обзорный** (выручка по категориям тариф/товар/услуга и способам оплаты нал/безнал/онлайн, бонусы, загрузка, топ позиций), **по клиентам** (дата регистрации, скидка, баланс, средний чек), **по посетителям** (уникальные, новые, гости vs клиенты), **по занятости оборудования** (рабочие часы хоста, выручка с ПК), **по играм и приложениям**, **по продажам** (разбивка по категориям).

## 9. Логи
Доступ: `admin.smartshell.gg/logs`. Фильтры: период (по умолчанию месяц), категория, переключение «все события / текущая смена». Структура события: **субъект** (кто) — **объект** (над кем) — действие — время. Категории: манипуляции с питанием ПК; продажа (сумма, скидки, промокоды); отмена продажи; сеансы (старт/конец/пересадка); изменения БД (клиенты, сотрудники, тарифы, товары, услуги, скидки, ПК); авторизация сотрудников; кассовая смена (начало/конец, наличные). Универсальная строка поиска: телефон, название товара, имя ПК.
