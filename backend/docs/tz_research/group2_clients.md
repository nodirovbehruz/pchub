# Группа 2: Клиенты

## 1. Клиенты

**Назначение.** Раздел `admin.smartshell.gg/clients` для работы со списком пользователей клуба, редактирования технического состояния, баланса и персональной скидки. База клиентов единая для всей сети SmartShell, но в каждом клубе у клиента свои условия: депозит, бонусный баланс, скидка. Клиент автоматически добавляется в список клуба после первой авторизации.

**UI и поля.** Главная страница: виджеты количества клиентов и групп, поиск (по логину/телефону), фильтр по группам, виджет суммы всех депозитов, кнопка выгрузки в .csv (только владелец). Таблица столбцов: никнейм, телефон, дата последнего посещения, персональная скидка, группа, депозит, статус, дата регистрации. Двойной клик открывает карточку клиента: краткая статистика, история покупок (дата, оператор, наименование, способ оплаты, сумма), редактируемые поля депозита и скидки, комментарии (могут пометиться важным — отображаются в окне продажи с восклицательным знаком), абонементы.

**Состояния и бизнес-логика.** Статус клиента (активен/заблокирован). Удалять комментарии может только Менеджер. Группы клиентов: создание (название до 16 Unicode-символов, скидка 0–100%), редактирование, удаление (только пустых). Клиент состоит в одной группе. Персональная и групповая скидки не суммируются — применяется наибольшая.

## 2. Регистрация и восстановления пароля клиента

**Назначение.** Самостоятельная регистрация клиента и сброс пароля из клиентского приложения (Шелл) для RU-региона.

**Поля формы регистрации.** Дата рождения; далее: никнейм, пароль, пароль повторно, номер телефона, e-mail, чекбоксы согласия с Пользовательским соглашением и ПОПД. **Кнопки.** «Войти», «Зарегистрироваться», «Далее», «Подтвердить звонком». **Логика подтверждения.** Бесплатный исходящий звонок на указанный номер. Если звонок невозможен — через 1 минуту доступны альтернативы: входящий звонок с кодом, Telegram-аккаунт, привязанный к номеру, VK по тому же номеру.

**Восстановление пароля.** «Войти» → «Забыли пароль?» → ввод телефона, нового пароля и повтора → «Подтвердить звонком».

## 3. Регистрация и восстановления пароля клиента не из RU-региона

**Отличия от RU.** Те же шаги и поля, но альтернативный метод подтверждения при невозможности звонка — **код в письме на e-mail** (вместо входящего звонка с кодом). Дополнительно сохраняются Telegram и VK. Все остальные шаги, бесплатный звонок, форма и единая база клиентов идентичны RU-флоу. Иные изображения экранов (nonRU-скриншоты).

## 4. Как изменить персональную скидку клиента

**Назначение.** Ручная корректировка персональной скидки клиента менеджером. **Шаги.** Панель управления → «Клиенты» → поиск по телефону/никнейму (минимум 4 символа) → двойной клик по строке → в карточке напротив скидки кнопка «Изменить» (иконка карандаша) → ввод значения → «Сохранить» (галочка). Учитывая бизнес-логику группы 1: персональная и групповая скидки не суммируются — применяется максимум.

## 5. Покупка игрового времени в Шелле

**Назначение.** Самостоятельная покупка тарифа клиентом через Шелл. Доступно в подписках **Starter и Business**, только зарегистрированным пользователям.

**UI.** Витрина тарифов сразу после входа. Показываются тарифы, разрешённые к продаже в клубе для категории клиента в текущее время. Самая левая серая карточка — активный тариф. **Шаги.** «Войти» → логин/пароль → клик по тарифу → окно подтверждения → информационное оповещение. Если тариф дороже депозита — Шелл показывает недостающую сумму. **Состояния и правила.** При выходе из аккаунта оставшееся время по всем тарифам, кроме абонемента, сгорает. Можно купить несколько одинаковых тарифов и несколько абонементов. Абонемент активируется сразу при покупке; приостановленный возобновляется на вкладке «Мои тарифы». При двух доступных абонементах следующий запускается автоматически, при трёх и более — клиент выбирает вручную. **Нельзя купить или активировать абонемент при активном тарифе.**
